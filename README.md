# Library Inventory Manager

A command-line based library inventory management system built with Python using Object-Oriented Programming principles.

## Features

- **Book Management**: Add new books to the inventory
- **Book Status Tracking**: Track whether books are available or issued
- **Search Functionality**: Search books by title or ISBN
- **File Persistence**: Save and load inventory data from text files
- **Exception Handling**: Robust error handling and logging
- **User-Friendly CLI**: Interactive menu-driven interface
